## 如何使用

将本仓库中 [engines.json](engines.json) 文件替换掉 `Zotero数据储存文件夹`\locate\engines.json 后重启Zotero即可。

**如何查看 `Zotero数据储存文件夹`位于哪里？**

依次点击 Zotero-编辑-设置-高级-数据储存位置
